C:\dev\adt-bundle-windows-x86_64-20130917\eclipse\plugins\org.apache.ant_1.8.3.v201301120609\bin



recon.cs
a utility to resize the windows console (dos box) to nearly full screen.
run again to toggle to 80x25 screen.
